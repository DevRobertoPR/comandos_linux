pwd navegación

Print Working Directory. Muestra la ruta completa del directorio en el que te encuentras actualmente dentro de la estructura de directorios.

cd navegación

Change Directory. Sirve para cambiar de directorio. Se usa con la ruta destino; por ejemplo cd /home.

cd ..Sube al directorio padre (anterior).
cd ~Va al directorio home del usuario.
.. navegación

Atajo que representa el directorio padre. Se usa junto con cd .. para retroceder un nivel en la estructura de directorios.

ls navegación

Lista el contenido del directorio actual: archivos y subdirectorios.

ls -aMuestra también archivos y carpetas ocultos.
ls -tOrdena por fecha de modificación.
ls -XOrdena por extensión de archivo.
ls -lLista detallada: usuario, grupo, permisos, tamaño, fecha.
ls -lhIgual que -l pero con tamaños legibles (KB, MB…).
mkdir archivos

Make Directory. Crea un nuevo directorio (carpeta) con el nombre que se indique. Ejemplo: mkdir proyectos.

rmdir archivos

Elimina un directorio vacío. Para usarlo hay que indicar la ruta del directorio o estar ubicado en su directorio padre.

touch archivos

Crea un archivo vacío o actualiza las marcas de tiempo de un archivo existente. La información de tiempo consta de tres atributos: tiempo de acceso, de modificación y de cambio.

rm archivos

Remove. Elimina archivos especificados en la línea de comando. Usado de forma recursiva también puede eliminar directorios completos.

rm -rElimina directorios y su contenido de forma recursiva.
rm -fFuerza la eliminación sin pedir confirmación.
clear utilidad

Limpia la pantalla de la terminal, eliminando el historial visual de comandos anteriores sin borrar el historial real.

chown permisos

Change Owner. Cambia el propietario de archivos, directorios y enlaces simbólicos. Se puede especificar usuario y grupo a la vez.

chown -RAplica el cambio de forma recursiva en todo el directorio.
useradd usuarios

Crea un nuevo usuario en el sistema. Sintaxis: useradd [opciones] nombre_usuario. Sin opciones extra crea el usuario sin contraseña ni directorio home.

mv archivos

Move. Mueve un archivo o directorio de un lugar a otro. También se usa para renombrar archivos. Ejemplo: mv archivo.txt /destino/.

mv -iPide confirmación antes de sobreescribir un archivo existente.
mv -uSolo mueve si el origen es más nuevo que el destino.
tar archivos

Tape Archive. Empaqueta y comprime una colección de archivos y carpetas. Genera archivos .tar; combinado con gzip produce .tar.gz.

-cCrear un nuevo archivo .tar.
-vMuestra el progreso detallado de la compresión.
-fEspecifica el nombre del archivo resultante.
-zCompresión gzip (.tar.gz).
-jCompresión bzip2.
-xExtraer el contenido del archivo.
-CExtraer en un directorio diferente al actual.
-rAgregar o actualizar archivos dentro de un .tar existente.
alternatives permisos

Establece enlaces simbólicos para programas predeterminados comunes. Permite que múltiples programas con funciones similares coexistan, determinando cuál se usa por defecto bajo un nombre genérico (ej. /usr/bin/editor). El administrador puede anular la selección automática en cualquier momento.

passwd usuarios

Asigna o cambia la contraseña de una cuenta de usuario. Ejemplo: passwd nombre_usuario. Pedirá la nueva contraseña dos veces para confirmar.
